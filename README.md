## 📋 Cardápio Digital

Este projeto consiste em um site de cardápio digital online, desenvolvido com o objetivo de apresentar produtos (como comidas, bebidas ou itens de um estabelecimento) de forma visual, organizada e interativa.

## 🚀 Objetivo do Projeto

O sistema foi criado para simular um cardápio moderno, facilitando a visualização dos itens disponíveis, seus preços e descrições, proporcionando uma experiência prática para o usuário — semelhante a soluções reais de cardápios digitais utilizadas por restaurantes e deliverys.

## 🖥️ Funcionalidades

- 📌 Exibição de produtos organizados por categorias
- 💲 Visualização de preços e descrições
- 🖼️ Layout visual atrativo com imagens
- 📱 Interface responsiva (adaptada para celular, tablet e desktop)
- 🎯 Navegação simples e intuitiva

## 🎨 Design e Interface

O projeto apresenta um design moderno e limpo, focado na experiência do usuário, com:

- Uso de cores harmoniosas
- Organização clara dos elementos
- Destaque para os produtos
- Layout inspirado em cardápios digitais reais

## ⚙️ Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦
- JavaScript 🟨

Essas tecnologias são amplamente utilizadas na criação de interfaces web interativas e responsivas.

## 📱 Responsividade

O site foi desenvolvido para funcionar perfeitamente em diferentes dispositivos, garantindo:

- Boa visualização em smartphones 📱
- Adaptação para tablets 📲
- Experiência completa em desktops 💻

## 💡 Importância do Projeto

Cardápios digitais são cada vez mais utilizados por estabelecimentos, pois:

- Facilitam o acesso aos produtos
- Melhoram a experiência do cliente
- Permitem atualização rápida de preços e itens
- Podem ser compartilhados facilmente por link ou QR Code
